## imagetools
### a command line tool for adding filters to images

#### Features
- Blur: `blur`
- Sharpen: `sharpen`
- Smooth: `smooth`, `smooth+`

#### TODO
- GUI (low priority)

#### Dependencies
- python3
- python3-pip

#### Instructions

```
git clone https://github.com/rahulrakida/imagetools.git
cd imagetools
pip3 install -r requirements.txt
python3 imagetools.py operation filename
```
